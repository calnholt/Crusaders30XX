MonoGame Implementation Notes

Trigger
Subscribe to PledgeAddedEvent in a new PledgeSlamAnimationSystem. Start anim when pledge is added; hand off to static PledgeDisplaySystem draw once complete.
Placement
Land position = card center + rotated offset (IconOffsetY * cardScale, default -190). Match IconScale (0.09) for final size. Animate position, scale, rotation with eased keyframes; card recoil via brief transform offset.
Impact FX
At impact frame (~380ms): expanding ring sprite, brief flash quad, dust puff below seal. Red border glow fades over ~600ms.
